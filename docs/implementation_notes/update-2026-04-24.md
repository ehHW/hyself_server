# 后端更新说明（2026-04-24）

本次更新聚焦系统维护机制、系统公告能力、直聊权限细化、上传大小限制与部署配置同步。

## 主要更新

- 新增系统维护中间件：
  - 新增 [auth/maintenance.py](auth/maintenance.py)
  - 在中间件链路中启用 [hyself_server/settings.py](hyself_server/settings.py)
  - 非超管在维护期访问接口会收到维护响应；超管保留操作入口

- 新增系统设置/公告领域能力：
  - 新增运行时逻辑 [hyself/system_runtime.py](hyself/system_runtime.py)
  - 新增系统 API 视图 [hyself/system_views.py](hyself/system_views.py)
  - 路由接入与兼容别名 [hyself/urls.py](hyself/urls.py)
  - 用户会话上下文改为读取运行时系统配置 [user/access_context.py](user/access_context.py)

- 新增系统数据模型与迁移：
  - 模型: [hyself/models.py](hyself/models.py)
  - 迁移: [hyself/migrations/0009_systemsetting_systemannouncement_and_more.py](hyself/migrations/0009_systemsetting_systemannouncement_and_more.py), [hyself/migrations/0010_systemsetting_announcement_content_max_length.py](hyself/migrations/0010_systemsetting_announcement_content_max_length.py)

- 直聊发送权限细化：
  - 规则：陌生人可发文本；仅“曾是好友后删除”禁发文本；陌生人仍禁发附件
  - 文件: [chat/domain/access.py](chat/domain/access.py), [chat/domain/friendships.py](chat/domain/friendships.py)
  - 覆盖测试更新 [chat/tests.py](chat/tests.py)

- 上传大小限制统一：
  - 在资源上传命令层新增 `UPLOAD_MAX_FILE_SIZE` 校验
  - 文件: [hyself/application/commands/resource_uploads.py](hyself/application/commands/resource_uploads.py), [hyself_server/settings.py](hyself_server/settings.py)

- WebSocket 维护态与系统事件增强：
  - 维护态连接判定 [ws/consumers.py](ws/consumers.py)
  - 全员系统事件与强制下线广播能力 [ws/events.py](ws/events.py)

- RBAC 权限补充：
  - 新增 `system.publish_announcement` 默认权限项 [user/signals.py](user/signals.py)

- 部署配置调整：
  - compose 中 backend/celery 镜像标签改为 1.1.0 [docker-compose.yml](docker-compose.yml)
  - `.gitignore` 忽略镜像归档文件 [*.tar, *.tar.gz] [.gitignore](.gitignore)

## 数据库迁移提醒

部署到生产环境前请确保执行：

```bash
python manage.py migrate
```

若是容器化部署并使用自动迁移入口，请确认容器启动日志中包含上述迁移执行记录。
